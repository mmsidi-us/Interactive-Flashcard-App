# Interactive Flashcard App

A lightweight, responsive, and interactive browser-based Flashcard application built with semantic HTML5, custom CSS3, and vanilla JavaScript. 

This project was built to practice core web development fundamentals, specifically focusing on DOM manipulation, state management, and event-driven JavaScript programming.

## 🚀 Features

- **Dynamic Q&A Cards:** Displays course-related questions one at a time.
- **Interactive Toggles:** Reveal answers dynamically with the click of a button.
- **Continuous Loop Navigation:** Smoothly cycle through the deck using the modulo operator (`%`), ensuring the deck automatically restarts after the final card.
- **Progress Tracking:** Real-time indicator displaying the user's current position (e.g., "Card 1 of 5").
- **Clean Responsive UI:** Center-aligned modern card interface built with responsive Flexbox, complete with micro-interactions like hover animations and active button scaling.

## 🛠️ Technologies Used

- **HTML5:** Semantic structural markup.
- **CSS3:** Custom styles, Flexbox layout, interactive hover transitions.
- **JavaScript (ES6+):** Event listeners, conditional rendering, state management, and DOM updating.
